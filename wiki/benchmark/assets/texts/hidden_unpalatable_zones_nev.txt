You will be prompted to define a new zone using the nevadoxes function in your chrome://settings/zone settings file.

For instance:

For all other zones, there will be a single defined zone that will be rendered in the middle with the name of the current browser zone set using the following syntax:

{ "namespaces": [ "org.geocities.browserzone", "org.geocities.nevadoxes", "org.geocities.nevadoxes-never1", "org.geocities.nevadoxes-neve1"], "nevadoxes": { "default": "yes", "resize": "smaller-size", "default": "yes", "path": "/zone"} }

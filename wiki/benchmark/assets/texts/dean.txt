Installation

After using this program, do the following:

sudo apt-get install libxml2

(this will install "LIBxml2/libxml2.so.7-version.dynamic.installer.packages.com" for DYNAMIC_EXTENSIONS)

Using Xorg

If you want to use the source code for Xorg, you just need to use the Xorg-related options:

sudo apt-get install libxml2

Note that if you download Xorg from a source server (eg. xorg-server.org), don't try to install it if it won't work on your system.

Once Xorg-based system is installed, you simply run /usr/sbin/Xorg -S .
